E-Commerce Website (MERN Stack)
 Overview :
This is a full-stack E-Commerce web application built using the MERN stack. It allows users to browse products, add items to cart, and place orders with secure authentication.

Features
🔐 User Authentication (Login/Register)
🛍️ Browse Products
🔍 Search & Filter Products
🛒 Add to Cart
💳 Place Orders
📦 Order Management
👤 User Profile
🛠️ Admin Panel (Add/Update/Delete Products)
